= Tags

Remember, when using tags, you have to add grooscript lib, and maybe jquery. Asset pipeline plugin is used.

[source,html]
--

<asset:javascript src="jquery.js"/>

    ... your grooscript tags here ...

<asset:deferredScripts/>
